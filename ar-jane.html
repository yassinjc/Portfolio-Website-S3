ALL THE CORE INFORMATION

For the group project in Semester 2, we designed a brandguide and website for the music artist Ar-Jane. Before we were able to design a complete website we made a brandguide which contained all the needed aspects of the style that'll be used in the website. Think about logo usage, the colorpalette, the fonts etc. After having finished the brandguide, it was time to make the website.

We started off in Adobe XD making various designs for the website. We kept in touch with Ar-Jane and went back and forth until we had a final design. We then continued to develop this design into a Wordpress website. The goal of this website was to be fully customisable for Ar-Jane so she could update it at any time.
